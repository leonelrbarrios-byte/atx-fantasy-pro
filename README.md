# ATX Fantasy EV Upgrade

This is a static front-end upgrade for your ATX Fantasy site.

## What's included
- NBA, MLB, and CS2 tabs in the scanner flow
- +EV scanner board
- Best odds cards
- Results page
- Pricing page
- Launch checklist page

## How to use
1. Upload these files into your existing site repository.
2. Replace your current `index.html`, `styles.css`, `script.js`, and `config.js`.
3. Commit the changes.
4. Vercel will rebuild automatically.

## For live odds later
You can either:
- replace the sample data in `script.js`, or
- point `config.js` to your own backend API and fetch real odds.
